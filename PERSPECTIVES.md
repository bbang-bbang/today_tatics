# Today Tactics — 관점별 사용자 분석

> K리그 전술 분석 도구를 바라보는 5가지 시선

---

## A. 현직 감독 (Head Coach)

**핵심 관심사:** 실전 활용 가능한 전술 도구인가?

### 잘하고 있는 것
- 실제 K리그 스쿼드 데이터 연동 → 선수 이름/번호가 실제와 일치해 소통 비용 절감
- 화살표(실선/점선/곡선/꺾기) 다양화 → 압박 루트, 오버래핑, 세트피스 동선 구분 가능
- 애니메이션 재생 기능 → 훈련 전 선수에게 동선 시각화 가능

### 아쉬운 점 / 개선 요청
- **포메이션 프리셋 부재** — 4-3-3, 3-4-3 등 포메이션 버튼 한 번으로 선수 배치 완성되면 실용성 ↑
- **세트피스 모드** — 코너킥/프리킥 전용 구역 강조(호/원 표시) 필요
- **상대 압박 시뮬레이션** — A팀 움직임 → B팀 반응 순서를 단계별로 재생하는 "시퀀스 모드"
- **저장 공유** — 이미지/PDF 내보내기 없으면 코치진 공유가 불편함
- 히트맵 데이터가 있다면 **실제 경기 기반 포지셔닝 제안** 기능도 유용

### 한 줄 평
> "전술 스케치 도구로는 충분한데, 훈련 현장에서 바로 쓰려면 포메이션 프리셋과 공유 기능이 먼저다."

## B. K리그 매니아 (K-League Fanatic)

**핵심 관심사:** 경기 이해도와 팬 경험을 높여주는가?

### 잘하고 있는 것
- K1 + K2 전 팀 망라 (29팀), 엠블럼과 팀 컬러까지 반영 → 팬으로서 소속감 충족
- HOME/AWAY 킷 구분 → 실제 경기 색상 그대로 시각화
- H2H 데이터 탑재 → 더비 매치, 강등권 6포인트 게임 분석에 활용

### 아쉬운 점 / 개선 요청
- **경기 일정 연동** — 이번 라운드 매치업이 홈 화면에 바로 떠야 진입 장벽이 낮아짐
- **팬 커뮤니티 공유** — "내가 만든 전술" 링크 공유 기능 → 커뮤니티(에펨코리아, 아프리카) 바이럴
- **선수 스탯 팝업** — 선수 아이콘 클릭 시 이번 시즌 득점/어시스트/출전시간 표시
- 2026 시즌 신규 팀(안양·부천 K1 승격) 팬이 특히 많이 쓸 것 → **승격/강등 히스토리** 탭 추가
- 중계 없는 지방 구단 경기 → **팬 전술 리뷰 게시판** 붙이면 커뮤니티 허브화 가능

### 한 줄 평
> "K리그 팀 데이터 완성도는 최고, 경기 일정 연동과 공유 기능만 추가되면 팬들 성지가 된다."

## C. 축구 현업 선수 (Professional Player)

**핵심 관심사:** 내 역할과 움직임을 직관적으로 파악할 수 있는가?

### 잘하고 있는 것
- 실제 등번호·이름으로 선수 표시 → "내가 어디 서 있어야 하는지" 바로 보임
- 드래그로 포지션 자유 배치 → 감독이 실시간으로 설명하며 조작 가능
- 애니메이션 재생 → 훈련 전 개인 동선 머릿속에 입력하기 편함

### 아쉬운 점 / 개선 요청
- **개인 강조 모드** — "나(특정 선수)만 하이라이트" 기능으로 멀티 플레이어 혼선 방지
- **롤 태그** — 선수 아이콘에 ST/CM/IWB 같은 롤 레이블 붙이는 옵션
- **반복 재생 + 루프** — 같은 동선을 반복 보며 익히는 용도 (현재 재생은 1회성)
- **태블릿/모바일 최적화** — 라커룸에서 폰으로 보는 경우가 많음, 터치 UX 개선 필요
- 히트맵을 **개인 히트맵**으로 세분화 → "지난 경기 내 활동 반경" 피드백 도구

### 한 줄 평
> "감독이 설명할 때 옆에서 같이 보면 이해는 된다. 혼자 복습하기엔 개인 강조 모드가 없어서 아쉽다."

## D. 스포츠 도박 매니아 (Sports Betting Enthusiast)

**핵심 관심사:** 경기 결과 예측에 실질적인 인사이트를 주는가?

> ⚠️ 이 관점은 도구의 데이터 분석 기능을 베팅 의사결정에 활용하려는 사용자 행동 패턴을 파악하기 위한 것입니다.

### 잘하고 있는 것
- **H2H 데이터** — 과거 상대전적, 홈/어웨이 구분 → 핸디캡 분석 기초 자료
- **팀 스탯(kleague_team_stats)** — 득점력/실점률 비교로 오버/언더 판단 참고
- **히트맵** — 공격 루트가 특정 사이드에 몰리는지 확인 → 세트피스 위험도 추정
- Sofascore 연동 데이터 → 유럽 베팅 사이트와 같은 소스 기반

### 아쉬운 점 / 개선 요청
- **폼 지표 시각화** — 최근 5경기 W/D/L 흐름이 팀 배너에 바로 보이면 좋겠음
- **부상/출전 가능 여부** — 현재 스쿼드는 정적 데이터. 실시간 결장자 정보 없으면 반쪽짜리
- **예상 라인업** — 감독 전술 선호도 기반 확률적 선발 예측 기능
- **홈/어웨이 보정 스탯** — 원정 팀 실점률은 홈과 다름, 분리 통계 필요
- 오즈 비교 탭 → 외부 배당률과 내부 데이터 갭이 벌어진 경기 자동 플래깅

### 한 줄 평
> "H2H와 팀 스탯은 쓸 만하다. 부상자 정보와 최근 폼이 실시간으로 붙으면 진짜 분석 도구가 된다."

## E. 전술 코치 (Tactical Coach / Assistant Coach)

**핵심 관심사:** 감독의 전술 철학을 선수에게 정확하게 전달하고 분석할 수 있는가?

> 감독이 "무엇을 할지"를 결정한다면, 전술 코치는 "어떻게 구현할지"를 설계하고 실행한다.

### 잘하고 있는 것
- 다양한 화살표 타입(실선/점선/곡선/꺾기) → 압박 강도·방향·우선순위를 화살표 종류로 구분 표현 가능
- A/B 양팀 동시 배치 → 상대 진형 대비 우리 팀 포지셔닝 대조 분석 가능
- 히트맵 오버레이 → 상대 팀의 공간 점유 패턴 파악 후 역습 루트 설계에 활용

### 아쉬운 점 / 개선 요청
- **레이어 시스템** — "1차 압박 라인 / 2차 커버 / 공 뺏은 후 전환" 등 단계별 레이어를 분리해 켜고 끄는 기능
- **전술 노트 첨부** — 각 화살표·포지션에 텍스트 메모를 달아 코치진끼리 의도 공유
- **영상 싱크** — 실제 경기 클립 옆에 전술판을 나란히 띄우는 분할 화면 (or 타임스탬프 연동)
- **상황별 전술 묶음** — "전방 압박", "5-4-1 수비 블록", "역습 패턴 A" 등 상황별 세트로 저장·전환
- **선수 역할 지시선 색상 구분** — 볼 캐리어 / 런 / 블록 / 마크 대상을 색으로 명확히 분리
- 히트맵을 **상대 팀 특정 선수 기준**으로 필터링 → 에이스 봉쇄 플랜 설계에 직접 활용

### 한 줄 평
> "전술 스케치 + 히트맵 조합은 분석 출발점으로 좋다. 레이어와 노트 기능이 붙으면 코치진 협업 도구로 진화할 수 있다."
